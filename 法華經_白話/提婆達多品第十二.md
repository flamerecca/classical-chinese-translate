# 妙法蓮華經　提婆達多品第十二

　　那時，佛陀告訴諸位菩薩以及天人四眾說：「我在過去無量劫中，為了求取《法華經》，從未懈怠疲倦。在許多劫中，我常常身為國王，發願求取無上菩提，心志始終不退轉。為了圓滿六波羅蜜，我勤奮行持布施，心中毫無吝惜，無論是象、馬、七珍、國土、城邑、妻子、奴婢、僕從，乃至頭目、骨髓、腦髓、身肉、手足，都不吝惜軀命。當時世間的人們壽命無量，我為了求法，捨棄了王位，將政事委付太子，擊鼓宣令，向四方徵求：『誰能為我宣說大乘，我將終身供其驅使。』

　　那時有一位仙人前來，對王說：『我有大乘之法，名為《妙法蓮華經》，若你不違背我，我便為你宣說。』王聽聞仙人之言，歡喜雀躍，當即隨侍仙人，供給其一切所需，採摘果實、汲取泉水、拾取柴薪、備辦飲食，乃至以自身為床座，身心從未疲倦。如此恭敬奉事，歷時千年，為了求法，精勤侍奉，令其無有缺乏。」

　　那時，世尊欲重新宣述此義，便說偈頌道：

我憶過去諸劫中，為求廣大妙法故，
雖然身為世間王，不貪五欲諸安樂。
擂鼓宣令告四方，誰人有此廣大法？
若能為我親解說，我當甘願為奴僕。

那時有一阿私仙，前來稟告大國王：
我有微妙殊勝法，世間稀有罕聞見，
若能依法修行者，吾當為汝善宣說。
時王聽聞仙人語，心中頓生大喜悅，

隨即便隨仙人行，供給其所有所需。
採薪拾柴並摘果，依時恭敬奉獻與。
心存妙法本願故，身心始終無懈倦。
普為廣大諸眾生，勤勤懇懇求大法，

亦非為了己自身，亦非為求五欲樂。
因此身為大國王，勤勤求法終獲得，
從此終得成佛道，今日特為汝等說。

　　佛陀告訴諸位比丘：「那時的國王，就是我本人。那時的仙人，就是今日的提婆達多。因為提婆達多是善知識的緣故，使我得以圓滿六波羅蜜、慈悲喜捨，以及三十二相、八十種好、紫磨金色、十力、四無所畏、四攝法、十八不共法、神通道力，成就無上正覺，廣度眾生，這一切都是由於提婆達多這位善知識的緣故。」

　　佛陀又告訴四眾說：「提婆達多在此後經過無量劫，當得成佛，佛號為天王如來、應供、正遍知、明行足、善逝、世間解、無上士、調御丈夫、天人師、佛、世尊。其國土名為天道。那時，天王佛住世二十中劫，廣為眾生宣說妙法，恒河沙數量的眾生證得阿羅漢果，無量眾生發起緣覺心，恒河沙數量的眾生發起無上道心，獲得無生法忍，達到不退轉的境地。」

　　「天王佛般涅槃之後，正法住世二十中劫。以其全身舍利建造七寶塔，高六十由旬，縱廣四十由旬，諸天人民，皆以各種鮮花、末香、燒香、塗香，以及衣服、瓔珞、幢幡、寶蓋，伎樂、歌頌，禮拜供養這座七寶妙塔。無量眾生證得阿羅漢果，無量眾生覺悟成辟支佛，不可思議數量的眾生發起菩提心，達到不退轉的境地。」

　　佛陀告訴諸位比丘：「在未來世中，若有善男子、善女人，聽聞《妙法蓮華經》提婆達多品，以清淨之心信敬，不生疑惑者，不墮地獄、餓鬼、畜生道，能生於十方佛前，所生之處，常能聽聞此經。若生於人天之中，受勝妙安樂；若生於佛前，則蓮華化生。」

　　此時，從下方多寶如來世尊隨行而來的菩薩，名叫智積，向多寶佛稟白說：「應當還歸本土了。」釋迦牟尼佛告訴智積說：「善男子，且稍等片刻，這裡有一位菩薩名叫文殊師利，可以與他相見，共論妙法，然後再返回本土。」

　　那時，文殊師利菩薩坐於大如車輪的千葉蓮華之上，同來的諸菩薩也坐在寶蓮華上，從大海之中的娑竭羅龍宮自然湧現出來，停住於虛空之中，詣靈鷲山而來，從蓮華上下來，到達佛所，以頭面禮敬二位世尊的足部。禮敬完畢，前往智積菩薩處，互相慰問，退坐於一旁。

　　智積菩薩問文殊師利說：「仁者前往龍宮，所教化的眾生，其數量有多少？」文殊師利說：「其數量無量，不可稱計，口所不能宣說，心所不能測量，且稍等片刻，自然當能親自印證。」話音未落，無數菩薩坐於寶蓮華上，從海中湧出，詣靈鷲山，停住於虛空之中。這些菩薩，都是文殊師利所教化度脫，具足菩薩行，皆共論說六波羅蜜。原本是聲聞人的，在虛空中宣說聲聞行，如今皆修行大乘空義。文殊師利對智積說：「在海中教化的事業，情形就是這樣。」那時，智積菩薩以偈頌讚歎說：

大智勇健功德廣，教化度脫無量眾，
今日此間諸大會，及我皆已親眼見。
演說暢達實相義，開示闡揚一乘法，
廣泛引導諸眾生，令其速速成菩提。

　　文殊師利說：「我在海中，只是常常宣說《妙法蓮華經》。」智積問文殊師利說：「此經甚深微妙，是諸經中的珍寶，世間稀有，是否有眾生能夠勤加精進、修行此經，迅速成佛？」文殊師利說：「有娑竭羅龍王之女，年僅八歲，智慧利根，善知眾生的諸根與行業，得陀羅尼，諸佛所說甚深秘藏，悉能受持。深入禪定，通達諸法，於一刹那之間，發起菩提心，得不退轉，辯才無礙。慈念眾生，如同對待自己的赤子，功德具足，心念口演，微妙廣大，慈悲仁讓，志意和雅，能夠到達菩提之境。」

　　智積菩薩說：「我看釋迦如來，在無量劫中，難行苦行，積累功德，求取菩提道，未曾停息。縱觀三千大千世界，乃至沒有如芥子許大的地方，不是菩薩捨棄身命之處，都是為了眾生的緣故，然後才得成就菩提道。我不相信此女能在須臾之頃，便成就正覺。」話未說完，那時龍王之女忽然出現在前，以頭面禮敬，退住一旁，以偈頌讚歎說：

深達罪福諸相義，遍照十方無所遺，
微妙清淨之法身，具足三十二相好，
以八十種種妙好，莊嚴無上之法身。
天人戴仰皆歸服，龍神無不咸恭敬，

一切眾生諸類別，無不宗奉而欽承。
又聞成就菩提道，唯有諸佛方能知，
我今宣揚大乘教，廣度苦難諸眾生。

　　那時，舍利弗對龍女說：「你說不久便能得無上道，此事難以令人相信。為何如此呢？女身垢穢，不是法器，如何能得無上菩提？佛道廣大深遠，需經無量劫、勤苦積累修行，具修各種波羅蜜，然後才能成就。再說女人之身，還有五種障礙：第一，不能成為梵天王；第二，不能成為帝釋；第三，不能成為魔王；第四，不能成為轉輪聖王；第五，不能成為佛身。如此，女身如何能迅速成佛？」

　　那時，龍女有一顆寶珠，價值等同三千大千世界，持以獻上佛陀。佛陀即刻接受。龍女對智積菩薩、尊者舍利弗說：「我獻寶珠，世尊納受，這事迅速否？」答言：「非常迅速。」龍女說：「以你們的神力，觀看我成佛，比這還要迅速。」

　　就在當時，眾會大眾皆見龍女，忽然之間，變成男子，具足菩薩行，即刻前往南方無垢世界，坐於寶蓮華上，成就無上正覺，具足三十二相、八十種好，普為十方一切眾生演說妙法。

　　那時，娑婆世界的菩薩、聲聞、天龍八部、人與非人，皆遙見彼龍女成佛，普為當時的天人大眾說法，心中大歡喜，皆遙相禮敬。無量眾生聽聞法義而解悟，得不退轉；無量眾生得受道記；無垢世界六次震動；娑婆世界三千眾生住於不退轉地，三千眾生發起菩提心而得受道記。智積菩薩及舍利弗，以及一切眾會，默然信受。
